Atributos HTML


Os atributos HTML fornecem informações adicionais sobre os elementos HTML.


Todos os elementos HTML podem ter atributos Os atributos fornecem informações adicionais sobre os elementos Os atributos são sempre especificados na tag inicial Os atributos geralmente vêm em pares nome / valor como: nome = "valor"
